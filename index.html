1. HTML y Estructura

HTML, o Lenguaje de Marcas de Hipertexto, es el lenguaje fundamental para construir páginas web. Utiliza etiquetas para definir la estructura y el contenido de un documento. Es esencial para determinar cómo se mostrará la información en la web, desde el texto hasta las imágenes y enlaces, y proporciona adaptabilidad para diferentes tipos de contenido. HTML es la base sobre la cual se construye la experiencia visual en Internet.

  • Etiquetas básicas: <html>, <head>, <body>, <title>, <meta>
  • Contenido: texto, imágenes, videos, formularios, tablas
  • Estructura semántica: <header>, <nav>, <section>, <article>, <footer>

Ejemplo:

<!DOCTYPE html>
<html>
  <head><title>Mi Página</title></head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Bienvenido a mi sitio web</p>
  </body>
</html>

2. CSS y Estilos

CSS (Cascading Style Sheets) controla el diseño visual de las páginas. Permite separar la estructura (HTML) del diseño (colores, márgenes, tamaños, animaciones).

  • Selectores: por etiquetas, clases, IDs
  • Propiedades comunes: color, background, margin, padding, display
  • Modelos de caja: cada elemento se representa como una caja con bordes, márgenes y relleno

Ejemplo:

body {
  background-color: #f0f0f0;
  color: #333;
  font-family: Arial;
}
h1 {
  color: #00838f;
}

3. JavaScript y Comportamiento

JavaScript es un lenguaje de programación que permite hacer que las páginas web sean interactivas. Se ejecuta en el navegador del usuario y permite manipular el contenido en tiempo real.

  • Eventos: click, mouseover, keydown, etc.
  • Funciones: bloques de código reutilizables
  • DOM: Document Object Model, estructura que representa el contenido de la web

Ejemplo:

document.getElementById("miBoton").onclick = function() {
  alert("¡Hola desde JavaScript!");
};

4. Diseño Responsivo

El diseño responsivo permite que las páginas se adapten automáticamente a diferentes tamaños de pantalla. Esto es crucial para ofrecer una buena experiencia en móviles y tablets.

  • Media queries: reglas CSS que aplican estilos según el ancho de pantalla
  • Frameworks: Bootstrap, Tailwind, Foundation
  • Unidades flexibles: %, em, rem, vw, vh

Ejemplo de media query:

@media screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

5. Accesibilidad Web

La accesibilidad web permite que personas con discapacidades puedan navegar, entender e interactuar con los sitios web de forma efectiva. Es esencial para la inclusión digital.

  • Etiquetas ARIA: atributos especiales para mejorar la comprensión de lectores de pantalla
  • Contrastes adecuados: facilita la lectura de texto
  • Teclado navegable: usuarios deben poder usar tab, enter y flechas

Ejemplo: <img src="imagen.jpg" alt="Descripción alternativa">
